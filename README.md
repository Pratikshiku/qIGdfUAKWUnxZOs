## Civil Service Exam Forum Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/36ee51337eb3483b8dd6b62d2d53b582/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Personal Center (modify personal information, change password)
4. Delivery Address
5. Product Information
6. Product Order History
7. Homepage
8. Experience Exchange Platform
9. Post Exchange
10. Comments and Messages
11. Announcement Information Viewing
12. Flea Market
13. Product Purchase


**Administrator Functions:**
1. Login
2. Registration
3. Personal Center (change password, modify personal information)
4. Administrator Management
5. Basic Data Management (post type management, announcement type management, product type management)
6. Experience Exchange Platform Management
7. Announcement Information Management
8. Flea Market Management
9. User Management
10. Carousel Information Management
